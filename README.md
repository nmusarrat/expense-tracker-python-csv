# expense-tracker-python-csv
A simple command-line expense tracker built with python and csv

##Features:
-Add expenses
-View expenses
-Calculate total spending
-Search expenses
-Delete expenses
-Show highest expense
-Saves data using CSV

##How to run:
1.Make sure python is installed
2.Run 'expense tracker csv.py'
3.Data will be saved in 'expenses.csv'(can be opened in Excel)
